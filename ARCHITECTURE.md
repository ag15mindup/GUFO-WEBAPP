# GUFO Rainbow Architecture

GUFO Rainbow is built as a modular fintech backend.

Core components:

User System
Manages user accounts and authentication.

Wallet System
Each user has a wallet that stores GUFO rewards.

Transactions
Tracks payments, cashback rewards and transfers.

Membership Levels
Users are assigned levels such as Bronze, Silver and Gold.

Database
Supabase (PostgreSQL) manages the core data.

Goal:
Create a scalable reward ecosystem that integrates with partner businesses.
