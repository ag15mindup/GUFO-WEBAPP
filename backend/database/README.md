# GUFO Rainbow Crypto

GUFO Rainbow is a cashback and rewards ecosystem designed to integrate digital wallets, membership levels and token-based incentives.

The project explores a fintech model where users receive rewards when interacting with partner businesses.

## Features

- Digital wallet system
- Cashback rewards
- Membership levels (Bronze, Silver, Gold)
- Transaction tracking
- Partner business integration

## Tech Stack

- Node.js
- Supabase
- PostgreSQL
- REST API
- JavaScript

## Architecture

The backend manages:

- Users
- Wallets
- Transactions
- Membership levels

Database logic is handled with Supabase.
